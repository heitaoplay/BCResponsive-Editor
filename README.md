# BCResponsive 编辑器

纯前端静态工具，用于**批量编辑 [BCResponsive](https://github.com/dynilath/BCResponsive) 插件**的动作（响应）与人格数据。

- 🌐 在线使用：https://heitaoplay.github.io/BCResponsive-Editor/
- 📦 技术栈：React + TypeScript + Vite，零后端，数据只在你的浏览器内处理。
- 🔗 与插件的压缩代码格式**双向兼容**。

## 功能

1. **导入 / 导出**：粘贴插件导出的压缩代码、原始 JSON、人格数组或全套设置；编辑后导出可粘回插件的代码（默认保留黑名单）。
2. **全面编辑**：人格、响应（动作）、消息的创建 / 修改 / 删除 / 上下移排序；触发器支持互动活动、高潮、兴奋度、进出聊天四种模式；活动 / 身体部位带智能提示且允许自由输入。
3. **合并**：多套人格的响应可追加、按名称合并或覆盖；支持合并为新人格。
4. **从零创建**：无需导入，直接新建人格并逐步添加响应与消息。
5. **响应式**：桌面与移动端均可操作。

## 与插件的兼容说明

- 网站导出的单人格代码 = `LZString.compressToBase64(JSON.stringify({name, responses, blackList}))`，
  与插件 `PersonaImportScreen` 的导入逻辑完全对应（插件本身也会保留 `blackList`）。
- 插件原生导出会丢弃 `blackList`，本工具在导出时额外保留它；若数据再次经插件导出则按插件逻辑重置（插件自身限制）。
- 数据校验逻辑复刻自插件 `V2ValidatePersonality`，非法数据会被拦截并给出友好提示。

## 本地运行

```bash
npm install
npm run dev      # 开发预览 http://localhost:5173
npm run build    # 产出 dist/ 静态文件
```

## 使用流程

1. 游戏内 BCResponsive → 人格设置 → 某人格「导入/导出」，复制文本框里的代码。
2. 在本工具「导入」页粘贴并解析。
3. 在「编辑」页批量修改；需要合并多套时用「合并」页。
4. 「导出」页选择人格，复制代码或下载文件。
5. 回到插件对应人格的「导入/导出」框，清空原内容、粘贴、确认。

## 隐私

本工具完全在浏览器本地运行，不会上传任何数据到服务器。
